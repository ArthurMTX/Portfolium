# Dashboard Layout Templates

This folder contains pre-configured dashboard layout templates for Portfolium.

## ğŸ¯ Built-in Templates

These templates are **built directly into the application**! You don't need to manually import them.

### How to Use Built-in Templates:

1. **Via Dashboard:**
   - Open the Dashboard page
   - Click the "Layouts" button (purple button in the top right)
   - Go to the **"Templates"** tab
   - Browse the available templates
   - Click "Load" on any template to apply it instantly

2. **Manual Import (Optional):**
   - If you want to customize before importing, copy the JSON from this folder
   - Go to the "Import JSON" tab in the Layout Manager
   - Paste the JSON
   - Click "Import Layout"
